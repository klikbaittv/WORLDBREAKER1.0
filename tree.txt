spellbook_for_writers/
================================================================

  00_spine_of_the_world.txt   THE authority. AI reads this first. All rules live here.
                               Load this in every session.

  01_how_to_use.txt           Onboarding guide. Start here if you're new.

  02_latest_save.txt          Pointer to your most recent save.
                               Always mirrors the highest-numbered save_file.

  03_characters.txt           Character registry. Append-only.

  04_world_notes.txt          Setting and world details. Append-only.

  05_lexicon.txt              World glossary — terms, places, slang, factions. Append-only.

  06_story_arc.txt            Plot structure and chapter tracking. Append-only.

  07_ledger.txt               Living canon document. NOT append-only.
                               Tracks current world state — who's alive, what's true.
                               Edit freely. The ledger reflects NOW.

  tree.txt                    This file. Project map.
